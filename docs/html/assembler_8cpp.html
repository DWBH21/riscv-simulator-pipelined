AssembledProgram assemble ( const std::string &  filename)

Assembles the intermediate code into machine code.

This function takes a vector of intermediate code blocks and assembles them into machine code. It uses the functions generateRTypeMachineCode, generateITypeMachineCode, generateSTypeMachineCode, generateBTypeMachineCode, generateUTypeMachineCode, and generateJTypeMachineCode to generate the machine code for each block.
