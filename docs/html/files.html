 main_assembler.cppIsolated Assembler for Testing. It reads a .s file, assembles it and converts the AssembledProgram struct into a file that can be decoded by the different VMs. This file logs the memory_controller_.write operations that the VmBase::LoadProgram functions performs to populate the text and data segments in memory The AssembledProgram is too complex to store directly into a file. This approach takes advantage of the fact that the assembly of the program and then loading it into memory is identical for all VM modes
